Markdown语法学习

Markdown中文版语法说明

标题

//# 一级标题//## 二级标题//### 三级标题
以此类推，总共六级标题，建议在井号后加一个空格。

列表

在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格。

引用

只需要在文本前加入 > 这种尖括号（大于号）即可

图片与链接

图片与链接
插入链接与插入图片的语法很像，区别在一个 !号

图片为：[]()

链接为：[]()

粗体与斜体

用两个*包含一段文本就是粗体的语法，用一个*包含一段文本就是斜体的语法。

表格

| title1 | title2 | title3 | | -------- |:--------:| --------:| | context1 | context2 | context3 | | context1 | context2 | context3 | | context1 | context2 | context3 |

代码框

1要用两个把中间的代码包裹起来。使用tab键即可缩进。

分割线

分割线的语法只需要三个 * 号
